# PHPMVC
A super fast, customizable and lightweight PHP MVC Starter Framework to extend for your own...

---

## How to Start
Clone this repo -
```sh
git clone https://github.com/ManiruzzamanAkash/phpmvc.git
```

## Modify the `.env` file
Duplicate `.env.example` and create `.env` file for your own -
```sh
BASE_DIR=

DB_HOST=localhost
DB_PORT=3306
DB_USER=root
DB_PASSWORD=
DB_NAME=phpmvc
```
